## 系统要求

<Info>
  **浏览器**: Chrome 90+
  **网络**: 稳定的互联网连接
</Info>

## 安装步骤

目前插件通过 GitHub 发布，请按照以下步骤安装：

<Steps>
  <Step title="下载插件">
    从 GitHub 下载对应的插件文件：
    - [EdgeX-Variational [001]](/products/edgex-variational) - 支持 EdgeX 和 Variational 平台
    - [Lighter-EdgeX [002]](/products/lighter-edgex) - 即将推出
  </Step>

  <Step title="解压文件">
    将下载的 zip 文件解压到桌面或其他便于访问的位置
  </Step>

  <Step title="安装到 Chrome">
    1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
    2. 右上角开启 **开发者模式**
    3. 点击 **加载已解压的扩展程序**
    4. 选择解压后的插件文件夹
    5. 确认安装完成
  </Step>
</Steps>

## 使用前准备

安装完成后，您需要：

<Steps>
  <Step title="激活产品">
    使用邀请码激活插件，详见 [产品激活](/getting-started/activation)
  </Step>

  <Step title="配置平台">
    配置交易平台账户：
    - [EdgeX 平台配置](/platforms/edgex-setup)
    - [Variational 平台配置](/platforms/variational-setup)
  </Step>

  <Step title="设置安全警告">
    配置饭碗告警系统，详见 [安全警告配置](/platforms/security-alerts)
  </Step>
</Steps>

## 验证安装

安装完成后，请确认：

<Note>
  - 插件图标显示在浏览器工具栏
  - 点击图标可以打开插件界面
  - 插件状态显示正常
</Note>

## 常见问题

如果安装过程中遇到问题：

1. **插件无法安装**：检查浏览器版本，清除缓存后重试
2. **插件无法加载**：确认已开启开发者模式，重新选择文件夹
3. **其他问题**：查看 [常见问题](/faq) 或联系技术支持
