## 系统要求

<CardGroup cols={2}>
  <Card title="浏览器插件" icon="puzzle-piece">
    - **浏览器**: Chrome 90+ 或 Edge 90+
    - **网络**: 稳定的互联网连接
    - **系统**: Windows / macOS / Linux
  </Card>
  <Card title="客户端程序" icon="desktop">
    - **系统**: Windows 10+ / macOS 11+
    - **网络**: 稳定的互联网连接
    - **硬盘**: 至少 100MB 可用空间
  </Card>
</CardGroup>

## 选择安装方式

根据您选择的产品类型，安装方式有所不同：

<Tabs>
  <Tab title="浏览器插件">
    适用产品：001, 002, 003, 005, 013, 014

    <Steps>
      <Step title="获取插件">
        访问对应产品页面获取下载链接：
        - [[001] EdgeX-Variational](/products/edgex-variational)
        - [[002] Lighter-EdgeX](/products/lighter-edgex)
        - [[003] Lighter-Based](/products/lighter-based)
        - [[005] Lighter-Backpack](/products/lighter-backpack)
        - [[013] Backpack-Variational](/products/backpack-variational)
        - [[014] Lighter-Variational](/products/lighter-variational)
      </Step>

      <Step title="解压文件">
        将下载的 zip 文件解压到便于访问的位置
      </Step>

      <Step title="安装扩展">
        1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
        2. 右上角开启 **开发者模式**
        3. 点击 **加载已解压的扩展程序**
        4. 选择解压后的插件文件夹
        5. 确认安装完成
      </Step>
    </Steps>
  </Tab>

  <Tab title="客户端程序">
    适用产品：010

    <Steps>
      <Step title="获取客户端">
        访问产品页面获取下载链接：
        - [[010] Lighter-Aster](/products/lighter-aster)
      </Step>

      <Step title="安装程序">
        1. 下载对应系统的安装包
        2. 运行安装程序
        3. 按照安装向导完成安装
      </Step>

      <Step title="启动程序">
        安装完成后，启动客户端程序
      </Step>
    </Steps>
  </Tab>
</Tabs>

## 使用前准备

安装完成后，按照以下步骤配置：

<Steps>
  <Step title="选择策略">
    从 [产品总览](/products/index) 了解不同策略的特点和磨损率
  </Step>

  <Step title="激活产品">
    使用邀请码（免费）或月费订阅（168 美元/月）激活，详见 [产品激活](/getting-started/activation)
  </Step>

  <Step title="配置平台">
    根据选择的策略配置对应的交易平台：
    - 查看 [支持平台](/platforms/index) 列表
    - 配置交易所账号和 API 密钥
  </Step>

  <Step title="设置安全告警">
    配置饭碗告警系统保护资产，详见 [安全警告配置](/platforms/security-alerts)
  </Step>
</Steps>

## 验证安装

安装完成后，请确认：

<Note>
  - 插件图标显示在浏览器工具栏
  - 点击图标可以打开插件界面
  - 插件状态显示正常
</Note>

## 常见问题

如果安装过程中遇到问题：

1. **插件无法安装**：检查浏览器版本，清除缓存后重试
2. **插件无法加载**：确认已开启开发者模式，重新选择文件夹
3. **其他问题**：查看 [常见问题](/faq) 或联系技术支持
