## 系统要求

在开始安装之前，请确保您的系统满足以下要求：

<Info>
  **浏览器**: Chrome 90+ / Edge 90+ / Brave  
  **操作系统**: Windows 10+, macOS 10.14+, Linux  
  **网络**: 稳定的互联网连接  
  **内存**: 建议 4GB 以上
</Info>

## 安装步骤

<Steps>
  <Step title="访问 Chrome Web Store">
    打开 Chrome 浏览器，访问 [Chrome Web Store](https://chrome.google.com/webstore) 
    
    或直接访问 AlphaLabs 插件页面
  </Step>
  
  <Step title="搜索 AlphaLabs">
    在搜索框中输入 "AlphaLabs Trading Plugin" 或 "AlphaLabs"
    
    找到官方发布的插件（认准官方认证标志）
  </Step>
  
  <Step title="添加至 Chrome">
    点击 "添加至 Chrome" 按钮
    
    在弹出的权限确认窗口中，点击 "添加扩展程序"
  </Step>
  
  <Step title="等待安装完成">
    插件将自动下载并安装
    
    安装完成后，您会在浏览器右上角看到 AlphaLabs 图标
  </Step>
</Steps>

## 初始配置

### 1. 打开插件

点击浏览器右上角的 AlphaLabs 图标，首次打开会显示欢迎页面。

### 2. 连接钱包

<Warning>
  请确保使用安全的网络环境连接钱包，避免在公共 WiFi 下操作。
</Warning>

支持的钱包类型：
- MetaMask
- WalletConnect
- Coinbase Wallet
- 其他 Web3 钱包

### 3. 选择交易平台

从支持的交易平台列表中选择您常用的平台：

  </Card>
  <Card title="OKX" icon="circle">
    综合性数字资产交易平台
  </Card>
  <Card title="Bybit" icon="arrow-trend-up">
    专注衍生品交易
  </Card>
  <Card title="更多平台" icon="plus">
    持续增加支持
  </Card>
</CardGroup>

### 4. API 配置

<Tabs>
  <Tab title="自动配置">
    1. 点击 "自动配置" 按钮
    2. 按照提示登录交易所账户
    3. 授权 AlphaLabs 访问权限
    4. 系统自动完成 API 配置
  </Tab>
  <Tab title="手动配置">
    1. 登录交易所，创建 API Key
    2. 设置权限：读取、交易（不需要提现权限）
    3. 复制 API Key 和 Secret
    4. 在插件中粘贴并保存
  </Tab>
</Tabs>

## 权限说明

AlphaLabs 插件需要以下权限才能正常工作：

| 权限 | 用途 | 必需性 |
|------|------|--------|
| 存储 | 保存配置和交易数据 | 必需 |
| 网络请求 | 与交易所 API 通信 | 必需 |
| 通知 | 交易提醒和告警 | 可选 |
| 标签页 | 显示交易界面 | 必需 |

<Note>
  AlphaLabs 不会收集您的私钥或 API Secret，所有敏感数据都在本地加密存储。
</Note>

安装完成后，请进行以下验证：

<Checklist>

  - [ ] API 连接测试通过
  - [ ] 可以获取实时市场数据
</Checklist>

## 故障排除

### 插件无法安装

1. 检查浏览器版本是否满足要求
2. 清除浏览器缓存后重试
3. 尝试使用隐身模式安装

### API 连接失败

1. 检查 API Key 是否正确
2. 确认 API 权限设置正确
3. 检查网络连接是否正常
4. 确认交易所服务状态

### 插件运行缓慢

1. 关闭其他不必要的标签页
2. 禁用其他可能冲突的插件
3. 重启浏览器
4. 检查系统资源使用情况

## 更新插件

AlphaLabs 插件会自动更新，您也可以手动检查更新：

1. 打开 Chrome 扩展管理页面 (`chrome://extensions/`)
2. 开启 "开发者模式"
3. 点击 "更新" 按钮
