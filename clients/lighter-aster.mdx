## 客户端简介

Lighter-Aster 010 是 AlphaLabs 的官方客户端应用，为用户提供统一的交易插件管理和监控界面。

## 系统要求

### Windows 系统
- **操作系统**：Windows 10 或更高版本
- **架构**：x64 (64位)
- **内存**：至少 4GB RAM
- **存储空间**：至少 500MB 可用空间

### macOS 系统
- **操作系统**：macOS 10.15 (Catalina) 或更高版本
- **架构**：Intel x64 或 Apple Silicon (M1/M2)
- **内存**：至少 4GB RAM
- **存储空间**：至少 500MB 可用空间

## 核心优势：极致去同化

Alpha Labs（Aster-Lighter版本）做到了极致的去同化，其中包括：

- **支持8个币种**：映射出28种不同的组合数
- **杠杆随机化**：每个用户设置杠杆不同，样本特征不同
- **波动率**：不同的波动率平仓，样本特征无任何相似特征
- **开单数量**：开单数量在区间内随机，每一笔都是不同的开单数量
- **开单时间随机**：在时间区间内随机下单，每一笔下单时间也完全不同

Alpha Labs做到了最大差异化，但是官方的女巫规则是一个黑盒。

## 防女巫建议

<Warning>
**重要提醒**：虽然我们做到了最大差异化，但仍需注意以下建议
</Warning>

### 减少女巫风险的建议

1. **币种轮换**：不要一直对冲同样两个币种，建议隔一段时间随机更换其他币种
2. **手动交易**：自动运行一段时间策略之后，建议可以手动去做一点合约单
3. **参数更新**：策略配置参数建议每隔一段时间更新一次（杠杆数、币种、平仓波动率）
4. **使用静态住宅IP，使用静态住宅IP，使用静态住宅IP**（重要事情说三遍）

### 已知的官方女巫规则

- MM（做市策略）根据官方的DC发布过相关消息会被女巫

## 安全警告

<Warning>
**运行环境要求**
</Warning>

1. **设备隔离**：客户端运行时，请保持和敏感设备隔离

2. **定期检查**：此版本客户端并未添加告警机制，运行策略时，请定期检查仓位情况
   - Aster和Lighter会因为内存机制，导致页面崩溃
   - 防止出现意外情况

3. **设备状态**：客户端运行情况下
   - 设备不可以关机
   - 设备不可以休眠
