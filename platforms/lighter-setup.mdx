## Lighter注册

1. 点击链接：https://app.lighter.xyz/trade/ETH?referral=JGD26BD4WLGK

    <img src="/images/lighter/002.png" alt="Lighter 平台首页" />

2. 点进 Transfer 进行存款（支持 **`Arbitru、Ethereum、Base、Ava`** 链 **`USDC`**）
**建议测试存款金额：最少100 usdc，后续按需进行存款**

    <img src="/images/lighter/003.png" alt="Lighter Transfer 存款" />

## API 配置

1. 点击Lighter前端页面Tools，点击 API Keys

   <img src="/images/lighter/004.png" alt="Lighter API Keys 页面" />

2. 如没有API Key则创建API Key，有API Key，点击 Refresh，然后保存 **`Public Key、Private Key`** 到本地。此处的 **`Private Key`** 就是 010 客户端内要填写的 Private Key

   <img src="/images/lighter/005.png" alt="创建 API Key" />

   <Note>
   如果在 Lighter 官网无法正常创建 API Key，可以使用 AlphaLabs 提供的便捷工具：https://lighter.keygen.alphalabs.app

   <img src="/images/lighter/007.png" alt="AlphaLabs Lighter Keygen 工具" />
   </Note>

3. 在Lighter页面，右键，点击检查，选择Application（应用），**`在Filter处输入 signature`**

4. 按图中箭头所指位置操作，看到**`Account index`**，复制下方箭头所指地方数字，图中所示为：**`2XXXX`**；记录并填入策略配置中的**`Lighter账户索引`**

   <img src="/images/lighter/006.png" alt="获取账户索引" />
