## API 配置

1. 点击Lighter前端页面Tools，点击 API Keys

   <img src="/images/lighter/004.png" alt="Lighter API Keys 页面" />

2. 如没有API Key则创建API Key，有API Key，点击刷新，**`一定要本地保存Public Key、Private Key`**

   <img src="/images/lighter/005.png" alt="创建 API Key" />

3. 在Lighter页面，右键，点击检查，选择Application（应用），**`在Filter处输入刚才复制的Public Key`**

4. 点击如图箭头所指位置，看到**`Account in   dex`**，填入策略Lighter账户索引

   <img src="/images/lighter/006.png" alt="获取账户索引" />

5. 记录账户 Account Index，后续在客户端配置时需要使用
