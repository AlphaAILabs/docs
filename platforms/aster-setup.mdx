# Aster 配置

## Aster注册

1. 点击链接：https://www.asterdex.com/zh-CN/referral/E01202    邀请id：**`E01202`
`将EOA地址，提交给@Cryptonymk 或 @DaPangDunCrypto，获取策略激活码（对应的激活码只能该eoa地址使用）`**

2. 点击右上角连接钱包，选择要连接的钱包种类

    <img src="/images/aster/001.png" alt="Aster 连接钱包" />

3. 点击 Deposit 进行存款UDSC、USDT等其他币种，可任意选择；支持 （Arbitrum、Ethereum、BNB chain）
温馨提示：BNB Chain选择USDF作为合约保证金，有20%的积分加成

    <img src="/images/aster/002.png" alt="Aster Deposit 存款" />

## API 配置

1. 打开Aster，点击更多，点击API管理

   <img src="/images/aster/003.png" alt="Aster API 管理" />

2. 点击创建API，复制**`API密钥`**，**`API秘密密钥`**（**一定要保存，只会明文显示一次**），保存在本地的txt中，策略配置时会进行使用

   <img src="/images/aster/004.png" alt="创建 Aster API" />
