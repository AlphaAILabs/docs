## 简介

通过配置 Telegram Bot，您可以实时接收策略运行状态和交易告警通知，确保及时了解账户动态。

<Info>
**Telegram 告警优势**：
- 实时推送消息通知
- 免费使用，无需额外费用
- 跨平台支持（手机/电脑）
- 消息记录可查询
</Info>

## 配置步骤

<Steps>
  <Step title="创建 Telegram Bot">
    在 Telegram 中搜索 **@BotFather**，点击左下角的 "Open"

    <img src="/images/notify/tg/001.png" alt="打开 BotFather" />

    点击 **Create a New Bot**

    <img src="/images/notify/tg/002.png" alt="创建新 Bot" />
  </Step>

  <Step title="设置 Bot 信息">
    按照提示输入 Bot 的名称和用户名，完成后点击 "Create Bot"

    <img src="/images/notify/tg/003.png" alt="输入 Bot 信息" />

    <Warning>
      **重要**：请妥善保管 Bot Token，这是您的机器人凭证
    </Warning>
  </Step>

  <Step title="启动 Bot">
    创建完成 Bot 之后，输入该 Bot 的 ID 去搜索该 Bot，然后点击 **Start** 按钮

    <img src="/images/notify/tg/004.png" alt="启动 Bot" />

    <Note>
      启动 Bot 后才能接收消息
    </Note>
  </Step>

  <Step title="获取 Chat ID">
    在 Telegram 中搜索 **@userinfobot**

    <img src="/images/notify/tg/005.png" alt="搜索 userinfobot" />

    发送任意消息

    <img src="/images/notify/tg/006.png" alt="发送消息" />

    记录返回的 Chat ID

    <img src="/images/notify/tg/007.png" alt="获取 Chat ID" />

    <Note>
      Chat ID 是您的 Telegram 账户唯一标识
    </Note>
  </Step>

  <Step title="配置到策略插件">
    将获取的 **Bot Token** 和 **Chat ID** 填写到策略插件的配置参数中

    **需要填写的信息**：
    - Bot Token（从第 2 步获取）
    - Chat ID（从第 4 步获取）

    <Warning>
      请确保 Bot Token 和 Chat ID 填写正确，否则无法接收告警消息
    </Warning>
  </Step>

  <Step title="测试告警">
    配置完成后，在插件中点击测试按钮，如果能正常收到 Telegram 消息即为配置成功

    <Note>
      如果未收到测试消息，请检查：
      - Bot Token 是否正确
      - Chat ID 是否正确
      - 是否已点击 Bot 的 Start 按钮
    </Note>
  </Step>
</Steps>

## 常见问题

<AccordionGroup>
  <Accordion title="无法创建 Bot 怎么办？">
    - 确保使用的是官方 @BotFather
    - 检查 Bot 用户名是否符合规则（必须以 bot 结尾）
    - 尝试使用不同的 Bot 用户名
  </Accordion>

  <Accordion title="收不到消息怎么办？">
    1. 确认已点击 Bot 的 Start 按钮
    2. 检查 Bot Token 和 Chat ID 是否正确
    3. 确认 Bot 没有被封禁
    4. 尝试重新获取 Chat ID
  </Accordion>

  <Accordion title="可以多个账户共用一个 Bot 吗？">
    可以。一个 Bot 可以给多个用户发送消息，只需要使用不同的 Chat ID 即可。
  </Accordion>

  <Accordion title="Bot Token 泄露了怎么办？">
    立即在 @BotFather 中重置 Bot Token：
    1. 找到对应的 Bot
    2. 选择 "Revoke token"
    3. 生成新的 Token
    4. 更新插件配置
  </Accordion>
</AccordionGroup>

## 安全提醒

<Warning>
**重要安全提醒**：
- Bot Token 是敏感信息，请勿泄露给他人
- 不要在公开场合分享 Bot Token
- 定期检查 Bot 的使用情况
- 发现异常立即重置 Token
</Warning>
