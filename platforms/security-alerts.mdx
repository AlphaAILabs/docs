## 配置步骤

<Steps>
  <Step title="访问告警网站">
    打开饭碗🥣告警网站：
    
    **网址**：https://fwalert.com/918826
    
    <Note>
      请确保使用正确的链接，避免访问钓鱼网站
    </Note>
  </Step>
  
  <Step title="选择提醒方式">
    在告警设置页面选择您偏好的提醒方式：
    
    <Tabs>
      <Tab title="电话提醒">
        - 适合紧急情况
        - 响应速度最快
        - 确保能及时接听
      </Tab>
      <Tab title="短信提醒">
        - 成本较低
        - 记录保存方便
        - 适合非紧急通知
      </Tab>
      <Tab title="双重保障">
        - 同时启用电话和短信
        - 最高安全级别
        - 推荐选择
      </Tab>
    </Tabs>
    
    配置完成后点击 **保存** 按钮
  </Step>
  
  <Step title="生成 Webhook">
    1. 在告警设置页面找到 **Webhook 配置** 部分
    2. 点击 **生成新的 Webhook**
    3. 复制生成的 Webhook URL
    4. 妥善保存此链接
    
    <Warning>
      Webhook URL 是敏感信息，请勿泄露给他人
    </Warning>
  </Step>
  
  <Step title="提供配置信息">
    将以下信息提供给技术支持团队：
    
    **需要提供的信息**：
    - Webhook 链接（从上一步获取）
    - 量化工具运行的 EOA 钱包地址
    
    **联系人**：
    - `@QuarticCat`
    - `@Cryptonymk`
    
    <Note>
      技术团队将帮助您完成最终的系统集成配置
    </Note>
  </Step>
</Steps>

  <Card title="交易异常" icon="exclamation-triangle">

    - 执行延迟异常

  <Card title="账户风险" icon="shield-alert">

    - 保证金不足

    - 异常登录

  <Card title="系统状态" icon="server">

    - API 调用失败
    - 网络连接异常
    - 系统性能问题
  </Card>
  <Card title="市场风险" icon="chart-line-down">
    - 极端价格波动
    - 流动性不足
    - 市场异常事件
    - 资金费率异常
  </Card>
</CardGroup>

## 配置建议

<Tabs>
  <Tab title="保守型">
    - 启用所有告警类型
    - 设置较低的阈值
    - 同时启用电话和短信
    - 适合新手用户
  </Tab>
  <Tab title="平衡型">
    - 启用重要告警
    - 设置中等阈值
    - 主要使用短信通知
    - 适合有经验的用户
  </Tab>
  <Tab title="激进型">
    - 仅启用紧急告警
    - 设置较高阈值
    - 主要依赖应用内通知
    - 适合专业交易者
  </Tab>
</Tabs>

## 常见问题

<AccordionGroup>
  <Accordion title="没有收到告警通知怎么办？">
    1. 检查手机号码是否正确
    2. 确认手机信号是否正常
    3. 查看是否被拦截软件屏蔽
    4. 联系技术支持检查配置
  </Accordion>
  
  <Accordion title="告警太频繁怎么办？">
    1. 调整告警阈值设置
    2. 关闭不必要的告警类型
    3. 设置告警时间间隔
    4. 优化交易策略参数
  </Accordion>
  
  <Accordion title="如何修改告警设置？">
    1. 重新访问告警网站
    2. 使用原有账户登录
    3. 修改相关设置
    4. 通知技术团队更新配置
  </Accordion>
</AccordionGroup>
