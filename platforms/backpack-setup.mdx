## 注册步骤

<Steps>
  <Step title="访问注册链接">
    点击链接：https://backpack.exchange/join/tezsnno5

    **邀请码**：`tezsnno5`

    <Note>
      Backpack 是一个一体化加密钱包和交易所
    </Note>
  </Step>

  <Step title="注册账号">
    填入谷歌电子邮箱，输入密码，并完成图形认证后进行注册

    <img src="/images/backpack/001.png" alt="Backpack 注册" />
  </Step>

  <Step title="激活账号">
    在谷歌邮箱处接收邮件，选择 **Sign in to Backpack Global**

    <img src="/images/backpack/002.png" alt="Backpack 激活邮件" />

    即可完成注册和登录操作
  </Step>

  <Step title="配置账号">
    新账号需要完成基础的账号配置

    1. 选择 **individual**（个人账户）

    <img src="/images/backpack/003.png" alt="Backpack 选择账户类型" />

    2. **KYC 认证**：根据自身情况按照要求填写完成 KYC 认证（Backpack 平台支持国人护照）

    <img src="/images/backpack/004.png" alt="Backpack KYC 认证" />

    完成后，返回设置页面

    <img src="/images/backpack/005.png" alt="Backpack 设置页面" />

    3. 在 setting 页面配置自己的语言选项和偏好

    <img src="/images/backpack/006.png" alt="Backpack 语言设置" />
  </Step>

  <Step title="账户充值">
    点击右上角的 **Deposit**

    <img src="/images/backpack/007.png" alt="Backpack 充值按钮" />

    选择币种及网络，之后向平台提供的地址进行充值即可

    <img src="/images/backpack/008.png" alt="Backpack 选择充值网络" />

    <Note>
      **建议测试存款金额**：最少 100 USDC，后续按需进行存款
    </Note>
  </Step>
</Steps>

## API 配置

<Steps>
  <Step title="登录账户">
    登录已注册的 Backpack 账户（未注册用户需通过 https://backpack.exchange/join/tezsnno5 注册）

    <img src="/images/backpack/009.png" alt="Backpack 登录" />
  </Step>

  <Step title="进入 API 设置">
    点击右上角账户总览，点击设置，点击 **API Keys**，点击新建 API Keys

    <img src="/images/backpack/010.png" alt="Backpack API 设置" />
  </Step>

  <Step title="创建 API Key">
    输入名称后，点击创建

    <img src="/images/backpack/011.png" alt="Backpack 创建 API Key" />
  </Step>

  <Step title="保存密钥">
    复制保存 **API Key** 以及 **API Secret**

    <img src="/images/backpack/012.png" alt="Backpack 保存密钥" />

    <Warning>
      API Secret 仅显示一次，请务必妥善保存
    </Warning>
  </Step>

  <Step title="配置 IP 白名单">
    设置 IP 限制，点击输入对应的 IP 地址

    <img src="/images/backpack/013.png" alt="Backpack IP 白名单" />

    <Note>
      配置 IP 白名单可以防止 API Key 被盗用
    </Note>
  </Step>
</Steps>
