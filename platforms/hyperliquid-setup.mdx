## 注册步骤

<Steps>
  <Step title="访问 Hyperliquid 平台">
    点击链接：https://app.hyperliquid.xyz

    <Note>
      Hyperliquid 是一个高性能的去中心化永续合约交易平台
    </Note>
  </Step>

  <Step title="连接钱包">
    1. 点击 **Connect Wallet** 按钮
    2. 选择您使用的钱包类型（推荐使用 Rabby 或 MetaMask）
    3. 在钱包中确认连接授权

    <Warning>
      确保您的钱包已切换到 Arbitrum 网络
    </Warning>
  </Step>

  <Step title="存款">
    1. 点击 **Deposit** 进行存款
    2. 选择存款币种：USDC
    3. 选择网络：Arbitrum
    4. 确认交易

    <Note>
      **建议测试存款金额**：最少 100 USDC，后续按需进行存款
    </Note>
  </Step>
</Steps>

## API 配置

<Steps>
  <Step title="生成 API Key">
    1. 登录 Hyperliquid 账户
    2. 前往 **Settings** > **API**
    3. 点击 **Create API Key**
    4. 设置 API 权限
    5. 保存 **API Key** 和 **Secret**

    <Warning>
      API Secret 仅显示一次，请务必妥善保存
    </Warning>
  </Step>

  <Step title="配置权限">
    根据需要设置 API 权限：
    - 交易权限
    - 查看权限
    - 提现权限（谨慎开启）
  </Step>
</Steps>
