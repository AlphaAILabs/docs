### 4. 使用程序策略

<Steps>
  <Step title="登录插件">
    打开插件，按照激活设置的各项参数登录插件

    <img src="/images/s_009/001.png" alt="登录插件" />
  </Step>

  <Step title="配置 API Key">
    接下来需要先配置好 API KEY

    <img src="/images/s_009/002.png" alt="配置 API Key" />

    参考以下教程获取并填写 API Key：
    - [Variational API 配置教程](/platforms/variational-setup#api-配置)
    - [Extended API 配置教程](/platforms/extended-setup#api-配置)

    填写完成后点击"保存密钥"

    <img src="/images/s_009/003.png" alt="保存密钥" />
  </Step>

  <Step title="配置策略参数">
    配置参数分为多个模块：

    <img src="/images/s_009/004.png" alt="配置策略参数" />

    **① 币种选择**

    选择您需要开合约的币种，两两为一对

    **② 杠杆设置**

    选择两个币种的杠杆倍数

    <Warning>
      **注意**：两个平台对应币种前端网页的杠杆倍数必须要和这里面设置的保持一样。

      当 API KEYS 填写完成之后，并且交易参数设置完成之后会自动去调整杠杆，但是需要用户在两个平台上查看杠杆修改是否生效。
    </Warning>

    **③ 波动率阈值**

    设定当对应币种相对于开仓价格波动多少则进行平仓

    - 波动值设置的小则开平仓频率会变高
    - 波动值设置的大则开平仓频率会变低

    **④ 交易时间段设置（机器所在时区）**

    开始时间：每天开启交易的起始时间

    结束时间：每天开启交易的结束时间

    比如：北京时间，7:30 ~ 22:30，策略将会只在这个时间段内进行交易，其他时间段将处于休眠状态

    **⑤ 最小持仓时间**

    最小值（分钟）：最小持仓时间随机数的最小值

    最大值（分钟）：最小持仓时间随机数的最大值

    比如：设置了 60min ~ 120min，则该 token 的最小持仓时间将在这中间去随机。为了防止爆仓，最小持仓时间和波动率阈值之间是 OR 的关系，也就是满足其中一个条件将触发平仓。

    **⑥ 每天可交易的最大交易数**

    最小值：每天可以交易的最大次数的随机最小值

    最大值：每天可以交易的最大次数的随机最大值

    开仓和平仓将分别计算一次。

    比如：XRP 开仓计算一次，平仓计算一次。假设每天只允许开 50 次，则之后不会再触发交易。

    **⑦ 拆单时间间隔**

    最小值（分钟）：拆单时间间隔随机数的最小值

    最大值（分钟）：拆单时间间隔随机数的最大值

    为了模拟真人的下单行为，拆单时间间隔将会是分钟级别的，由用户自行设定，防止 VAR 系统认为用户在攻击 OLP。

    **⑧ Extended 和 Variational 开仓 Size**

    设定对应币种单个开仓的数额范围，系统会根据设定在范围内随机数额开单，保证开仓数量的随机性

    **⑨ 批次运行随机时间区间**

    设置下单之后暂停多少秒再进行下一单的操作，增加时间上的随机性

    **⑩ 滑点设置**

    如果选择交易小币种则需要将点差设置到 0.02%—0.03%

    **Telegram 告警配置**

    策略启动、策略异常等情况发生之后，都会通知用户。

    参考：[Telegram 告警配置](/platforms/telegram-alerts)

    <img src="/images/s_009/005.png" alt="Telegram 配置 1" />
    <img src="/images/s_009/006.png" alt="Telegram 配置 2" />

    ---

    全部设置完成后，点击"保存设置"

    <img src="/images/s_009/007.png" alt="保存设置" />
  </Step>

  <Step title="启动策略">
    点击主页面上的"启动策略"，然后观察策略运行情况即可

    <img src="/images/s_009/008.png" alt="启动策略" />
  </Step>
</Steps>
