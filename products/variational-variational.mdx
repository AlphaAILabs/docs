### 2. 设置服务启动脚本
<Frame>
  <img src="/images/vultr/003.png" alt="选择 Startup Script" />

点击 `Add Startup Script`

<Frame>
  <img src="/images/vultr/004.png" alt="添加启动脚本" />

<Frame>
  <img src="/images/vultr/005.png" alt="填写脚本" />

比如使用名称 `variational-service-api`，下面的脚本直接复制填写：

```bash
#!/bin/bash
set -e

echo "=== Starting Variational Service Deployment ==="

# Install dependencies
apt-get update
apt-get install -y caddy ufw

# Download service
mkdir -p /opt/variational
cd /opt/variational
curl -L -o variational-service https://github.com/AlphaAILabs/variational-api-service/releases/download/v1.0.0/variational-service-linux-amd64
chmod +x variational-service

# Create systemd service
cat > /etc/systemd/system/variational.service << 'EOF'
[Unit]
Description=Variational API Service
After=network.target

[Service]
Type=simple
WorkingDirectory=/opt/variational
ExecStart=/opt/variational/variational-service
Restart=always

[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload
systemctl enable variational
systemctl start variational

# Configure firewall
ufw --force enable
ufw allow 22
ufw allow 80
ufw allow 443

# Get server IP and configure Caddy

SERVER_IP=$(ip -4 addr show | grep -oP '(?<=inet\s)\d+(\.\d+){3}' | grep -v '127.0.0.1' | head -1)

cat > /etc/caddy/Caddyfile << EOF
$SERVER_IP {
    reverse_proxy localhost:3000
    tls internal
}
EOF

systemctl restart caddy

echo "=== Deployment Complete ==="
echo "Access service at: https://$SERVER_IP"
```

### 4. 验证和测试

复制服务器 IP

<Frame>
  <img src="/images/vultr/017.png" alt="复制 IP" />
</Frame>

在浏览器中打开 `https://你的IP`。由于使用自建证书，Chrome 会告警。

**推荐方式：启动忽略证书的 Chrome**

<Tabs>
  <Tab title="macOS">
    打开终端，输入以下命令启动新的 Chrome 窗口：

    ```bash
    open -na "Google Chrome" --args --ignore-certificate-errors --ignore-urlfetcher-cert-requests --user-data-dir=/tmp/chrome-dev
    ```

    这会开启一个新的 Chrome profile，自动忽略证书验证。

    <Frame>
      <img src="/images/s_019/013.png" alt="macOS 终端命令" />
    </Frame>
  </Tab>
  <Tab title="Windows">
    1. 按 `Win+R` 输入 `notepad` 打开记事本
    2. 粘贴以下代码：

    ```bat
    @echo off
    start "" "C:\Program Files\Google\Chrome\Application\chrome.exe" --ignore-certificate-errors --ignore-urlfetcher-cert-requests --user-data-dir=%TEMP%\chrome-dev
    ```

    3. 点击"文件" → "另存为"
    4. 文件名输入：`chrome-dev.bat`（必须带 .bat 后缀）
    5. 保存类型选择：**所有文件 (\*.\*)**
    6. 保存位置选桌面，点击"保存"
    7. 双击桌面的 `chrome-dev.bat` 即可启动

    <Frame>
      <img src="/images/s_019/012.png" alt="Windows 创建 bat 文件" />
    </Frame>
  </Tab>
</Tabs>
