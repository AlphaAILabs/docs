## 产品概述

AlphaLabs Signer Manager 是本地签名服务管理器，提供安全、高性能的订单签名能力，配备现代化的图形化管理界面。

| 属性 | 信息 |
|------|------|
| **产品状态** | 已上线 |
| **支持平台** | macOS (Apple Silicon / Intel) / Windows (x64) |
| **GitHub** | https://github.com/AlphaAILabs/signer-manager |

## 核心特性

## 快速开始

<Steps>
  <Step title="下载安装">
    从 [GitHub Releases](https://github.com/AlphaAILabs/signer-manager/releases) 下载对应系统的安装包并安装
  </Step>

  <Step title="启动应用">
    打开 AlphaLabs Signer
  </Step>

  <Step title="启动服务">
    点击"启动服务"，等待初始化完成

    <img src="/images/signer-manager/001.png" alt="启动 Signer 服务" />
  </Step>

  <Step title="配置策略">
    在对应策略中配置服务地址：`http://localhost:10000`

    <img src="/images/signer-manager/002.png" alt="配置服务地址" />
  </Step>
</Steps>

<Note>
  可在设置中修改服务端口，修改后需同步更新策略配置
</Note>

## 重要提醒

<Warning>
  - 私钥仅存储在本地设备，请妥善保管并定期备份
  - 运行 Signer Manager 的电脑**不可休眠、不可关机**
  - 确保服务端口（默认 10000）未被占用
  - 不建议在公共或不受信任的设备上运行
</Warning>
