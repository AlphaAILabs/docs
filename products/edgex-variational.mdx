### 1. 下载策略程序

<Steps>
  <Step title="下载压缩包">
    下载策略程序压缩包：https://github.com/AlphaAILabs/perp-plugin-strategy-001/releases/download/v0.0.1/perps-edgex-variational-001.zip
  </Step>

  <Step title="解压缩">
    解压下载的压缩包文件

    <img src="/images/s_001/001.png" alt="解压缩文件" />
  </Step>

  <Step title="加载扩展程序">
    通过谷歌浏览器 → 扩展程序 → 管理扩展程序，加载刚刚解压缩的文件夹

    <img src="/images/s_001/002.png" alt="管理扩展程序" />
    <img src="/images/s_001/003.png" alt="加载已解压的扩展程序" />
  </Step>

  <Step title="检查加载状态">
    检查 001 策略已经正常加载

    <img src="/images/s_001/004.png" alt="检查加载状态" />
  </Step>
</Steps>

### 2. 配置 AlphaLabs 官网账号

要使用 AlphaLabs 的策略程序，需提前配置好官网账号：

<Steps>
  <Step title="连接钱包">
    在官网链接钱包（此钱包不需要是需要跑策略的钱包）
  </Step>

  <Step title="创建账户">
    点击"创建账户"，然后按照要求激活策略（请认真填写自己的信息，如果不正确则无法使用策略）

    访问：[https://alphalabs.app/products/activation](https://alphalabs.app/products/activation)
  </Step>

  <Step title="等待审核">
    AlphaLabs 的后台收到信息后，会有专门的 dev 进行审核处理，审核通过则可以使用策略
  </Step>
</Steps>

### 3. 使用程序策略

<Steps>
  <Step title="登录插件">
    打开插件，按照激活设置的各项参数登录插件

    <img src="/images/s_001/005.png" alt="登录插件" />
  </Step>

  <Step title="配置 API Key">
    接下来我们需要先配置好 API KEY

    参考以下教程获取并填写 API Key：
    - [EDGEX API 配置](/platforms/edgex-setup#api-配置)
    - [Variational API 的配置](/platforms/variational-setup#api-配置)

    <img src="/images/s_001/006.png" alt="配置 API Key" />

    填写完成后点击"保存密钥"
  </Step>

  <Step title="配置策略参数">
    配置参数分为 5 个模块（IM 和 MM 不要调整）：

    **① 最大价差**：平台间的最大 BTC 价差

    **② 平仓价差**：达到平仓价差才可执行平仓
    - 设置的小则开平仓频率会变高
    - 设置的大则开平仓频率会变低

    **③ 开仓数量**：单次开仓的最小数量

    **④ 杠杆倍数**：选择两个币种的杠杆倍数

    <Warning>
      **重要**：两个平台对应币种前端网页的杠杆倍数必须要和这里面设置的保持一样
    </Warning>

    **⑤ Gain 数值**：参考插件展示的 log 数值，无需频繁调整，波动过大时再进行调整

    <img src="/images/s_001/007.png" alt="配置策略参数" />

    全部设置完成后，点击"保存设置"
  </Step>

  <Step title="启动策略">
    点击主页面上的"启动策略"，然后观察策略运行情况即可
  </Step>
</Steps>
